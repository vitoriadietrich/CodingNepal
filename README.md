# Top 10 CSS-3 Neumorphism UI Designs | CodingNepal

Uma coleção de **10 projetos UI com efeito neumorphism** usando HTML, CSS e JavaScript.

## ✨ Sobre

Este projeto reúne **10 designs modernos em estilo neumorphism**, explorando diferentes componentes de interface com sombras suaves e efeito 3D.

O estilo **neumorphism (Soft UI)** combina design flat com sombras para criar um visual leve e “embutido” na interface :contentReference[oaicite:0]{index=0}.
